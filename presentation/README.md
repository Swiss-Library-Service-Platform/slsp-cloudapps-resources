# SLSP Cloud Apps Workshop Presentation

This folder contains the presentation materials for the SLSP Developer Forum workshop on Alma Cloud Apps.

## Files

- **[slides.pdf](slides.pdf)** - Built presentation (open with PDF viewer)
- **[slides.md](slides.md)** - Presentation content in Markdown format, easy to edit

## Technology

The presentation is built using [Marp](https://marp.app/) (Markdown Presentation Ecosystem), which converts Markdown to slides.

To rebuild PDF:
```bash
npx @marp-team/marp-cli slides.md -o slides.pdf
```
