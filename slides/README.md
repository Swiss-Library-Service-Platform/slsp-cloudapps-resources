# Workshop Slides

Presentation materials for the SLSP Cloud Apps Workshop.

## Files

- **[slides.pdf](slides.pdf)** - Presentation (open with PDF viewer)
- **[slides.md](slides.md)** - Source (Markdown format)

## Rebuilding

Uses [Marp](https://marp.app/) to convert Markdown to PDF:

```bash
npx @marp-team/marp-cli slides.md -o slides.pdf
```

To also export to PowerPoint (.pptx):
```
npx @marp-team/marp-cli slides.md -o slides.pptx
```
